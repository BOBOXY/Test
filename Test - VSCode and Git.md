# 在VSCode中使用Git

1 在一个目录下clone项目：
```
git clone XXXXXX.git
```

2 使用VScode 打开项目

3 使用vscode提交代码

3.1 进行更改

3.2 点击文件旁边的`+`；相当于`git add .`,这会将文件添加到**暂存的更改**部分，表示它将包含在下一次提交中。

也可以通过选择源代码管理视图中**更改**/**Changes**旁边的 **+**（加号）图标一次性暂存所有待处理的更改。
